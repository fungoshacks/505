CHANGELOG:

X-PROJECT 1.5.6
(PLAYSTATION 4 FIRMWARE: 5.05-5.07)

Customization:
[+] XMB Tinting: New STYLE column featuring 16.8 million colour picker, randomised or revert to default options.
Buttons and text also match this tint in all pages of X-Project.
[+] Avatars: Place your own 210x210px PNG in the root folder (INDEX.HTML location). A fall back USER.PNG is provided if none is found.
(When X-Project is loaded you can click on a colour within your avatar to colour your XMB to match).
[+] Name: Enter your name in the infomation panel to be greeted throughout.
[+] IP address: Enter the IP address of your console for quick reference. When using our HTTP server to host, it will resolve your IP automatically.
[+] Icons: Use your own 60x60px PNG in the ICONS folder and make relevant edits in INDEX.JS.
(See INDEX.JS for working examples).

New Features:
[+] DNS server method: enter the DNS provided and access X-Project in USERGUIDE or BROWSER (x.project).
[+] Spoofing payload: Choose your own custom version number 0.00-9.99.
[+] Fan Payload: Set your own fan temperature 40° min to 79° max.
[+] Kernel time: Use any of the REACTPSPLUS payloads to set the kernel time on your console, and have it display alongside the system time.
(WARNING: Deleting cache removes this data)
[+] Automatic Caching: If X-Project is not detected: it caches. It's highly recommended for stability.
[+] Automatic Update Caching: If X-Project detects an updated offlinexmb.cache: it updated automatically.
[+] DETAILS section in the ABOUT column for contacting the X-Project Team and making requests and suggestions.
